# Markdown → HTML 表示テスト

これは **GitHub Pages で Markdown から出力した HTML が正しく表示されるか** を確認するためのテストページです。

---

## 1. 見出しレベル

# 見出し h1
## 見出し h2
### 見出し h3
#### 見出し h4

---

## 2. 強調・装飾

- *イタリック*
- **ボールド（強調）**
- ***イタリック＋ボールド***
- ~~取り消し線~~
- `インラインコード`
